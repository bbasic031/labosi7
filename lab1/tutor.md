# Git tutor

``` 
git status
```

Prikazuje status repozitorija.

```
git add FILEPATH
```

Dodaje novu datoteku u repozitorij.

```
git commit -m "MESSAGE"
```

Commita datoteku.

```
git diff
```

Prikazuje promjene.

```
git log
```

Prikazuje sve dosadašnje commitove.

```
git clone PATH
```

Klonira repozitorij.

```
git pull
```

Povlači sa remotea lokalno.

```
git push
```

Pusha na gitlab repozitorij.
